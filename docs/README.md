## Config compatibility

`polyws` supports these workspace config filenames:

- `.polyws`
- `.poly`
- `.polyws.json`
- `.poly.json`
- `.polyws.toml`
- `.poly.toml`

Extensionless files (`.polyws`, `.poly`) support both JSON and TOML with auto-detection.
